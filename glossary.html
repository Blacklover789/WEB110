HTMl and CSS Glossary

Section-2 Html

Element
An element is a piece of content and the specific meaning it carries, as well as the code used to mark it up.
Tag
The code that delineates an element is called tags.
Paragragh
Default text is used when you code a Paragragh tag.
Self-closing element
there are some elements that don't need a closing tag because they don't wrap around any content.

Section-4 Skeleton code

Doctype
Lets the browser know which version of html document uses.
Title
It is what shows in the browser tab and in search listings on Google and each search engines.
Body
Elements that holds all content that is for display on the page itself.
Meta viewport
This code indicates that the browser should not zoom in on the page and should respect any responsive CSS code.

Section-5 Basic Html Elements

Headings
Section titles in an HTML documents.
Small
Represents text that is less important, side comment,copyright, or legal information.
Strong
An inline element used to add importance to text. By default and most commonly,it makes the text bold.
Unordered list
The ul element represents the list of items, where the order of the items is not important_that is, where changing the order would not materially change the meaning of the list.
